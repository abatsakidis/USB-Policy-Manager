## Description

USB Policy Manager 

## How To ##

* Run the application
* Select USB Policy Access Mode (Full/Read Only/Disabled) 
* Select Registry Editor Policy (Disable or not)
* Click Apply button

* Select "Set Password" for setting Password protection for the application
